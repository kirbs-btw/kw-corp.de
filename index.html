How to use ?

Step 1: (Impotant!)

Choose the folder where you want
your folders to be.
User Preferences > locate your projects folder > change settings

The way how your foders are named can be changed under:
User Preferences > Select Foldername type > change settings

Step 2:

To create a folder. Type in your content
and click create folder.
If you have existing folders just drag and drop
them in to the big white space!

Step 3:

To find your folders again go to
the search menu and type in your keyword
and click search.
To change the description just type in
a new one and press the save button.

Tricks:
To search for multiple keyword just put a ":"
between the tags
e.g.
chair:house
